## Pipeline
This part organizes codes for world-coordinate reconstruction, which integrates multiple thirdparty packages that have seperate licenses to follow. Please visit their websites for more detail on licenses. 
- [SAM2](https://github.com/facebookresearch/sam2)
- [GVHMR](https://github.com/zju3dv/GVHMR)
- [Detectron2](https://github.com/facebookresearch/detectron2)
- [DROID-SLAM](https://github.com/princeton-vl/DROID-SLAM)
- [Metric3D](https://github.com/YvanYin/Metric3D)
- [ViTPose](https://github.com/ViTAE-Transformer/ViTPose) 
- [SPEC](https://github.com/mkocabas/SPEC)
